This application is intended to reproduce erorr in connection between `webpack` and `react-multimedia-capture` described in https://github.com/rico345100/react-multimedia-capture/issues/2
## Reproduce error

To reproduce error run following commands and open `index.html` file:

```
git checkout master
npm install
npm run build
```

## Run working version

To enter applicaiton which depends on working fork of `react-multimedia-capture` (https://github.com/bartmalanczuk/react-multimedia-capture) run following commands and open `index.html` file:

```
git checkout solution
npm install
npm run build
```
